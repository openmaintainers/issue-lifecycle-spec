# Implementation Guide

## Suggested Labels

- status:new
- status:triaged
- status:accepted
- status:in-progress
- status:blocked
- status:needs-review
- status:resolved
- status:closed
- status:archived

## Recommended Practices

- Every issue starts as `new`
- Triage within 7 days
- Explicitly use `blocked`
- Archive stale issues after 180 days
- Maintain transition history for analytics
